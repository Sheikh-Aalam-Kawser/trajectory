# Software Requirements Specification (SRS) - Trajectory

## 1. Introduction
Trajectory is an AI Execution Companion designed for students, early-career developers, and internship seekers. It transforms vague goals into dynamic, self-healing execution plans, providing structured steps and active guidance.

## 2. Core Features
- **Planning Agent**: Breaks commitments down into highly actionable milestones.
- **Prioritization Agent**: Automatically schedules and arranges tasks based on urgency and priority.
- **Adaptation Agent**: Reschedules delayed tasks and guides the user back on track when they fall behind.

## 3. Technology Stack
- **Frontend**: React 19, Vite, Tailwind CSS, TanStack Query, React Router, Lucide Icons.
- **Backend**: Node.js, Express, Firebase Admin, Google GenAI (Gemini 2.0 Flash).
